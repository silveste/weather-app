This app has been developed by Antonio Fernandez to create a data visualization for the module CA682 Data Management and Visualisation of GCAI1 Grad Cert in Artificial Intelligence course (2018/2019)

The app uses Dark Sky and Nominatim APIs.

The map is an svg file obtanied in wikimedia commons and only the background color has been modified.

The weather icons have been created by Yun Liu and are available at www.iconfinder.com
All classes inside the icons has been removed and replaced with inline styles.

Usage:

  • When load the page wait until the icons appear in the map (this could take a few minutes)
  • Hover the icons to get more detailed information
  • Press the arrows to see other days forecast
  • Click at any point in the map to get information or search the city using the input box at the top (this could take a few minutes)
  • Remove any icon by clicking on it
